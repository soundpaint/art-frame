$ sudo apt-get install gimp

* Dia (for icons creation)

$ sudo apt-get install dia

* InkScape (for icons creation)

$ sudo apt-get install inkscape

==== Install art-frame ====

* Change to home directory:

$ cd

* Clone Git repository:

$ git clone https://github.com/soundpaint/art-frame.git

* Compile art-frame:

$ cd art-frame
$ make

* Create "bin" directory:

$ mkdir ~/bin

* Copy all shell script files from the unpacked art-frame directory to the
  ~/bin directory:

$ cp -p ~/art-frame/scripts/*.sh ~/bin/

==== X Autostart ====

* Make RasPi boot into text console with auto login (e.g. via
  raspi-config).

* Create "~/.bash_login":

$ cd ~
$ nano .bash_login

* Add the following lines to "~/.bash_login":

#!/bin/bash
if [ $(tty) == "/dev/tty1" ]; then
    bin/fan-init.sh
    bin/fan-on.sh
    startx /home/pi/bin/art-frame.sh
    sudo shutdown -h now
fi

* Disable Xsession screen saver by editing /etc/xdg/lxsession/LXDE/autostart:

  Comment out with comment character "#" (or remove the line completely):

  # @xscreensaver -no-splash

  Add the following lines:

  # art-frame: no screen saver
  @xset s noblank
  @xset s off
  @xset -dpms

* Disable terminal screen saver (optional):

  Edit file /etc/kbd/config:

  BLANK_TIME=0
  POWERDOWN_TIME=0

* Optionally: Rotate LCD screen by 180 degrees, useful for some Pi cases

  Append to file /boot/config.txt the following 2 lines:

  # art frame: rotate LCD screen by 180 degrees
  lcd_rotate=2

* Optionally: Disable touch (usually, you do *not* want to do this)

  Append to file /boot/config.txt the following 2 lines:

  # art frame: disable touch functionaly
  disable_touchscreen=1

==== Audio Configuration ====

$ sudo raspi-config
[select "9 Advanced Options" > "A8 Audio" > "1 Force 3.5mm ('headphone') jack"]
[select "Ok"]
[select "Finish"]

$ alsamixer
[select maximum volume]
[exit]

==== Images Configuration ====

Use the configuration file config.xml to specify which iamges to use as start-off.
Provide those images referred to in the config.xml file at the file path that is
specified in the configuration file.
